=== Root ===

Версия 2.2.4

== Установка ==

https://docs.wpshop.ru/themes/root/

Вариант 1.
1. Перейдите в раздел Внешний вид - Темы - Добавить новую - Загрузить тему
2. Выберите скачанный архив и нажмите Установить
3. Активируйте тему
4. Перейдите в раздел Настройки - Настройки темы и введите лицензионный ключ

Вариант 2.
1. Разархивируйте скачанный архив.
2. Поместите папку root в /wp-content/themes/
3. Перейдите в раздел Внешний вид - Темы и активируйте тему
4. Перейдите в раздел Настройки - Настройки темы и введите лицензионный ключ

== Рекомендованные плагины ==

* Q2W3 Fixed Widget - позволяет закреплять виджеты
* WP-PostViews - выводит колчество просмотров постов
* Yoast SEO - один из лучших СЕО плагинов
